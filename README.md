## Реализовано

- Авторизация пользователей
- Роли: администратор, модератор, пользователь
- Просмотр списка книг с пагинацией
- Добавление, редактирование и удаление книг
- Загрузка обложек
- Жанры книг
- Markdown-описание книг
- Рецензии пользователей
- Статусы рецензий: на рассмотрении, одобрена, отклонена
- Модерация рецензий
- Раздел "Мои рецензии"
